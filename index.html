Что такое программирование.
Зачем нужны языки программирования.

Программирование – процесс написания (создания) программ, в ходе которого с помощью специальных записей языка программирования, компьютер понимает какой алгоритм ему следует выполнить и соответственно выполняет при запуске программы.

Язык программирования – это формальный язык, предназначенный для написания компьютерных программ (т.е. инструкций алгоритмов, которые выполняет компьютер). Каждый язык программирования имеет свои лексические, синтаксические и семантические правила, обязательные, и те, которым можно не следовать. На данный момент таких языков больше восьми тысяч, но мы будем говорить про основные, с которых можно будет начать изучение.

Опытные программиста всегда знают несколько языков и порой даже работают с несколькими одновременно. Вы всегда сможете начать изучать более подходящий для ваших нужд язык. Более подробно тему языков программирования мы разберем далее.

Направления в программировании
В сфере программирования существует множество профессий, но направлений только три. Чтобы начать познавать какое-либо направление обычно изучают один или несколько языков нужных для старта, устоявшихся временем. Далее, рассмотрим каждое направление и узнаем о нужных для них знаниях. Важно отметить, что по мере изучения предоставленного материала следует интересоваться самим направлением, в котором вы развиваетесь, узнавать информацию о других языках, профессиях и т.д. Таким образом, вы поймёте со временем каким программистом вы хотите быть, какие языки и технологии вам для этого нужны, также вы уже сами откроете для себя новые источники информации.
1. Web-разработка

Веб-разработка представляет собой создание сайта или веб-приложения, но помимо кодирования в нее входит и изготовление дизайна. Я ознакомлю вас с веб-разработкой со стороны программирования.
Веб-разработка – это одно из самых легких направлений программирования. С помощью нее легче всего войти в сферу. Многие программисты-самоучки начинали с веб-разработки, а после уходили в другое направление.

В программирование сайта/веб-приложения входит:
  • Вёрстка макета дизайна. (Frontend разработка)
  • Программирование логической части сайта/веб-приложения (Backend разработка)
Frontend разработка

Frontend (‘Фронтенд’, англ. front-end) – лицевая часть веб-приложения, всё что вы видите на нем. Для разработка фронтенда, за исключением некоторых дополнительных технологий, всегда используют три языка.

HTML (англ. HyperText Markup Language, «язык гипертекстовой разметки») – язык разметки веб-страниц. Его используют для того, чтобы отметить на сайте различные его элементы (блоки) и после описать их внешний вид с помощью языка CSS.

CSS (англ. Cascading Style Sheets, «каскадные таблицы стилей») – язык описания внешнего вида веб-страницы. С помощью селекторов (англ.“selector” происходит от слова “select” – выбрать) выбирают HTML-элемент и изменяют его внешний вид.
Существует 3 уровня данного формального языка, это CSS1, CSS2, CSS3. они различаются технологиями и доступный функционалом. в данный момент самая актуальная версия – CSS3.

JavaScript (сокр. “JS”) – язык программирования с очень широким функционалом, используется не только в веб-программировании. Последние девять лет по рейтингам сервиса StackOverflow (сервис вопросов и ответов о программировании, очень популярен и имеет большое значение в развитии программирования) JavaScript является самым популярным языком программирования.
JS используется для создания динамической интерактивности сайта. Например: смена одной картинки на сайте на другую, проигрывание музыки по нажатии кнопки и т.п.
В 2012 году компания Microsoft представила новый язык программирования для веб-разработки под названием TypeScript. Предполагалось, что он расширит возможности JavaScript, но в итоге, TypeScript не опередил JavaScript по количеству использований в программировании сайтов.

Существуют расширения и фреймворки (различные дополнениях к языкам, библиотеки, облегчающие разработку). Они необязательны к изучению, особенно при старте. Следуя моему совету, вы сами поймёте, когда вам стоит заняться этим.

    Необязательно! Следует изучать, если вы хотите знать JS выше начального уровня

Backend разработка

Backend (‘Бэкенд’, англ. Back-end) – программная часть веб-приложения, отвечает за его программное (логическое) функционирование.

Примеры Бэкенд-элементов сайта: авторизация на сайте, сохранение информации в базу данных, получение сообщения от знакомого в социальной сети.

Сама бэкенд разработка обычно осуществляется с помощью языка программирования и базы данных. Я предлагаю вам изучить язык программирования PHP и базы данных MySQL. С этих двух технологий начинает большинство новичков программистов.

PHP (англ. Hypertext Preprocessor, «препроцессор гипертекста») – язык программирования общего назначения, в большинстве случаев применяемый в разработке веб-приложений. Преимущество PHP перед остальным языками программирования подходящими для бэкенда в том, что PHP изначально создавался только для бэкэнд разработки.

Почему PHP:
  • Много ресурсов для изучения языка, большое сообщество.
  • Не труден в изучении.
  • Легок в установке и использовании.
  • Изначально создавался только для бэкэнд разработки

MySQL – свободная система управления базами данных с широким функционалом.

Важно отметить, что сначала следует изучить язык программирования PHP и только после изучать MySQL.

Порядок изучения:
  1. Для изучения PHP по следующему курсу вам понадобится установить программу OpenServer. Вам поможет сделать это данная статья: *Ссылка*.
  2. *Ссылка* - данный курс хорошо обучит вас основам PHP.
  3. *Ссылка* - полное и более подробное обучение PHP. Можно начать изучение с этого курса и не использовать курс из второго пункта, но стоит учесть, что на курсе сайта metanit.com всё написано более профессиональным языком.
  4. *Ссылка* - обучение MySQL.
  5. *Ссылка* - обучение работе PHP с MySQL.
  6. *Ссылка* - использовать для справки.
2. Разработка компьютерных игр

Разработка компьютерных игр проходит поэтапно, и обычно занимается команда, состоящая из людей с различными должностями. Также, в команде присутствуют программисты, занимающиеся разными частями игры. Но их начальное обучение одинаково.

Для вашего дальнейшего развития в сфере разработки игр сначала следует изучить основы игрового движка и сам язык программирования, подходящий для этого движка.

Игровой движок - программное обеспечение использующееся, для создания игр. Он создает основу игры и после помогает добавить другие составляющие игры.

Я предлагаю вам начать с игрового движка Unity и языка программирования C#.

Unity бесплатный для изучения игровой движок. На нем созданы миллионы игр. В сообщество гейм-разработчиков присутствуют люди, которые плохо относятся к данному движку в виду его недостатков, но несмотря на это, изучения разработки игр все равно обычно начинают именно с него.

Почему Unity:
  • Бесплатен для изучения.
  • Большое сообщество, множество обучающих видеоуроков, статей.
  • Легче в изучении чем альтернативные игровые движки.

C# (“Си-шарп”, от англ. C-sharp) - язык программирования разработанный компанией Microsoft. Является единственным подходящим языком программирования для создания игр в Unity.

Порядок изучения:
  1. *Ссылка* - курс по языку C++, а не требующемуся C#, но вы должны пройти его для понимания основ программирования. Синтаксис у этих двух языков схож, курс по C# более трудный и требует начальных знаний.
  2. *Ссылка* - видеокурс по C#, познакомит вас с основами языка и не только.
  3. *Ссылка* - видеокурс на английском языке (для развития в сфере программирования вам обязательно понадобятся знания английского языка) по созданию видеоигры в Unity. Познакомитесь с основами игрового движка
  4. *Ссылка* - видеокурс, обучающий созданию реалистичной физики объектов игры в движке Unity
3. Разработка desktop/mobile приложений

Desktop/Mobile приложение (Desktop с англ. “Рабочий стол”, Mobile с англ. “Мобильный” (телефон)) – программа, которая устанавливается на устройство пользователя и работает под управлением операционной системы (ОС - комплекс программ, предназначенных для управлениями ресурсами устройства).
Если в будущем вы хотите заниматься разработкой программ машинного уровня (примеры: программа для управления реактором, трансформатором), то сначала вам следует научиться создавать Desktop-приложения и только потом идти дальше.

Такие приложения бывают разные по объему. Разработчиком может быть, как и один программист-любитель, так и большая команда из профессионалов.
В процесс познания данной сферы не входит ничего удивительного. Вам лишь нужно хорошо знать язык программирования и уметь использовать его в решении определенных задач.

В процесс познания данной сферы не входит ничего удивительного. Вам лишь нужно хорошо знать язык программирования и уметь использовать его в решении определенных задач.

Сначала вам следует определиться для каких устройств вы хотите создавать приложения. Также, определиться с операционной системой, для которой вы будете создавать приложения. Для телефонов это IOS или Android. Для персональных компьютеров Windows, MacOS или Linux.

Если вы хотите разрабатывать программы для Windows или Linux, то я предлагаю вам изучить C#. Важно отметить, что для того, чтобы создавать программы под Linux, соответственно, вы будете использовать во время разработки ОС Linux.

В случае с IOS, MacOS – язык программирования Swift. Существует две среды разработки для IOS, это XCode и AppCode. XCode бесплатен и доступен только для MacOS (операционная система для персональных компьютеров от компании Apple), AppCode доступен для других ОС, но платно, а также в нём отсутствует множество важных функций, которые есть в XCode.
Если перефразировать вышесказанное, то для разработки IOS приложений у вас должно быть устройство Mac (какой-либо из ПК от компании Apple), которое сможет поддерживать последнюю версию MacOS хотя бы два года.
Но при этом, у нас есть возможность попробовать использовать XCode даже на ОС Windows. Мы можем сделать это, установив виртуальную операционную систему на ваш ПК (статья, обучающая этому: *Ссылка*. Важно отметить, что в таком случае все программы будут работать очень медленно, работать/учиться в таких условиях невозможно.
